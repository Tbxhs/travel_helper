== 行程单

想出去玩，不知道大概要花多少钱？
旅行的时候遇到新朋友，想立刻分享给他下一步的行程安排？看看有没有交集？

此想法起源于我自己的旅行安排，纵观互联网文字图片攻略类网站多如牛毛，但没有一个可以帮助我规划自己的行程。
其实我只是想看看路线怎么走以及大概的花费，却不得不花费大量的时间耐着性子看完那些又臭又长充满“感悟”和骚气的攻略。
